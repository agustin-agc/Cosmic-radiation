En este proyecto quise construir una herramienta completa para analizar el riesgo de invertir en acciones.

Mi objetivo fue armar un flujo real y automático en donde primero se descargan los datos de internet para luego visualizarlos en un gráfico y tomar decisiones claras.

---

## 1. Obtención de Datos (ETL)

Creé scripts en Python que se conectan a internet, descargan los precios reales de las acciones y los limpian para que estén listos para usar.

## 2. Simulación Matemática

Programé un motor de **Monte Carlo** que proyecta cientos o miles de posibles futuros para una acción basándose en su comportamiento histórico.

## 3. Base de Datos

Diseñé una base de datos SQL donde guardo el historial de precios y los resultados de las simulaciones.

## 4. Dashboard Visual

Conecté todo a **Power BI**, transformando la matemática en visualizaciones claras para identificar acciones seguras o riesgosas.
